# MITRE-Threat-Research-Task-Internship-Task-1-
Project Title: MITRE ATT&CK Research Project - Campaign & Malware Analysis
​Overview:
This repository contains a comprehensive Cyber Threat Intelligence (CTI) research report prepared for the MITRE ATT&CK Research Internship Selection Program. The project demonstrates the ability to analyze advanced cyber threats and map adversary behaviors using the MITRE ATT&CK framework.
​Key Research Areas:
​Campaign Analysis (Campaign 00): In-depth analysis of the SolarWinds Supply Chain Compromise, attributing tactics, techniques, and procedures (TTPs) to APT29 (Cozy Bear). Includes attack flow, objectives, and defensive recommendations.
​Malware & Tool Mapping: Technical analysis of 10 specific software entries and malware strains (including Mimikatz, Poison Ivy, China Chopper, SeaDuke, and Zbot).
​Threat Intelligence: Detailed mapping of execution methods, persistence, privilege escalation, and Indicators of Compromise (IOCs) for each tool to assist in network defense and detection operations.
